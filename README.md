# 📚 SmartStudy-Desktop

> Uma aplicação voltada para a gestão de estudos, focada em otimizar o tempo e organizar a rotina acadêmica. 

## 💡 A Origem do Projeto
O **SmartStudy-Desktop** surgiu de um problema real: como estudante de ensino a distância (EAD), senti a necessidade de uma ferramenta própria para centralizar o controle de matérias, aulas e prazos. O objetivo é facilitar a organização da minha rotina acadêmica e garantir que nenhuma atividade seja esquecida.

---

## 🛠 Tecnologias Utilizadas
- **Electron**: Interface desktop.
- **Vue.js**: Framework frontend.
- **Node.js**: Backend e scripts.
- **Vite**: Ferramenta de build.

---

## ⚙️ Como Executar
Para rodar o projeto, execute os comandos abaixo em seu terminal na ordem indicada:

```bash
git clone [https://github.com/ntcruz/SmartStudy-Desktop.git](https://github.com/ntcruz/SmartStudy-Desktop.git)
